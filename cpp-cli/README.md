# cpp-cli

CLI para scaffolding de proyectos C++ con CMake.

## Instalación

1. Clona el repositorio
   git clone https://github.com/tu-usuario/cpp-cli.git

2. Agrega la carpeta al PATH de Windows

## Uso

cgen new project MiJuego
cgen new class engine/Player
cgen new module graphics/Renderer
cgen build
cgen run
cgen dist
